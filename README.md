# 2º Lista de exercício - PIF

Construir um sistema completo em trio ou quarteto, combinando: variáveis, printf/scanf, operadores, if/else e switch.

Escolha um tema:
1. Caixa Registradora: Ler produtos, calcular total, aplicar descontos por faixa.
2. Sistema de Notas: Ler notas, calcular média, classificar status, exibir relatório.
3. Calculadora Científica: Menu interativo com validações completas.
